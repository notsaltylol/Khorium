=======
Khorium
=======

An example Trame application


* Free software: Apache Software License


Installing
----------
Build and install the Vue components

.. code-block:: console

    cd vue-components
    npm i
    npm run build
    cd -

Install the application

.. code-block:: console

    pip install -e ".[dev]"


Run the application

.. code-block:: console

    Khorium
